Lockscreen Image Generator

How to use:  In the text box above, input a list of phrases that you want displayed. This could be positive affirmations, daily routine reminders, or betrayals by an ex‑partner.

 Select the width and height based on your device screen resolution. Use the Y-axis position and font size to position the text. Select your colors, and an optional background image.

 Generate the images and download them to your device. Rotate through the generated images using either built-in software (e.g., Apple or Samsung) or an app (e.g., Android).
